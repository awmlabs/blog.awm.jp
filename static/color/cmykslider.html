* CMYK => RGB
    R = 1 - min(1, C * (1 - K) + K)
    G = 1 - min(1, M * (1 - K) + K)
    B = 1 - min(1, Y * (1 - K) + K)

* RGB = CMYK
    K = 1 - max(R, G, B)
    if ( K == 1) {
        C, M, Y = 0
    } else {
        C = ( 1 - R - K) / (1 - K)
        M = ( 1 - G - K) / (1 - K)
        Y = ( 1 - B - K) / (1 - K)
    }
